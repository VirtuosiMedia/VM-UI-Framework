VM UI Framework Beta

A modern, grid-based CSS, layout, and UI framwork for HTML5, VM UI Framework was created to assist in creating websites and, more specifically, web applications. It's a companion to VM PHP Framework, but can be used independently as well. It's inspired by Twitter's Bootstrap framework, but we wanted to take it in a slightly different direction.

For Modern Browsers

In order to keep our codebase neat and tidy, we are only supporting the latest versions of each browser. Most browsers update themselves, but I know what you're wondering: this means IE9+.

MooTools-Powered

All of our JavaScript plugins use MooTools, a JavaScript framework that allows for fast, flexible, and clean development.
