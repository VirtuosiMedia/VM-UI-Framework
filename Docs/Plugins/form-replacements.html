Form Element Replacements

Overview

VM UI Framework offers automatic replacements for some of the tough-to-style form elements, allowing for a consistent and stylable interface across all browsers. All are completely keyboard accessible and should work as drop-in replacements for their regular form element counterparts.

Usage

As long as the form replacement plugins are loaded on the page, no extra code is required. All selects, checkboxes, and radios will automatically be replaced.

Note The select replacement plugin does not currently support multiple-choice selects and so any select element with the multiple attribute will not be converted.
