Form Element Replacements

VM UI Framework offers automatic replacements for all of the tough-to-style form elements, allowing for a consistent and stylable interface across all browsers. All are completely keyboard accessible and should work as drop-in replacements for their regular form element counterparts.

Code Syntax Highlighting

There are two types of syntax highlighting available: static and dynamic. Static will convert any <pre> elements into highlighted source code, while dynamic highlighting will convert a textarea into a live editor.
