Form Plugins

Auto-complete

Provides a list of options beneath an input while the user is typing. The options can be loaded via inline content or AJAX. Keyboard-accessible.

View Auto-complete Plugin

Form Replacements

Replaces checkboxes, radio buttons, and select dropdown elements with a styled replacement that maintains a consistent look across browsers.

View Form Replacements Plugins

Form Validation

Over 20 unique validations for all form elements, plus the ability to create custom validators. Form elements can be validated one at a time or when the form is submitted.

View Form Validation Plugin

Password Strength Meter

An entropy-based password strength meter, this plugin also checks the password against the top 1000 most-commonly used passwords.

View Password Strength Meter Plugin

Password Toggle

Allows a user to toggle the visibility of their password for verification.

View Password Toggle Plugin

Sliders

Creates text-based or numeric sliders. Sliders are compatible with keyboard input, scrollwheels, mouse input, or touch-based devices.

View Sliders Plugin

Layout Plugins

Baseline

Automatically adjusts margins of of select elements to conform to a vertical rhythm, readjusting as needed when the window is resized.

View Baseline Plugin

Modal Windows

Load content in a modal window from an iframe, AJAX, or inline content. Custom events allow you to take action after the modal is closed. 100% responsive.

View Modal Window Plugin

Pager

Automatically paginate inline content. The pager plugin will take existing pages and divide them into pages, showing only one at a time.

View Pager Plugin

Pinned Elements

Pin an element and it will remain in place as the user scrolls until it reaches the limits of its parent container.

View Pinned Elements Plugin

Slideshow Carousel

Create responsive slideshows and carousels, with 6 default styles and 7 transition types. Includes support for thumbnails, captions, and custom controls.

View Slideshow Carousel Plugin

Navigation Plugins

Accordion Menu

Turns any multi-level list into an accordion menu, with control over which sections are opened or closed at start.

View Accordion Menu Plugin

Dropdown Menu

Creates dropdown menus for top navigation or buttons. Also supports mega-dropdown menus that span the entire width of the grid. Menus are all 100% responsive.

View Dropdown Menu Plugin

Mobile Menu

Collapses a navbar menu and replaces it with a menu button that expands when clicked or touched.

View Mobile Menu Plugin

Tabs

Regular and pill tabs, with horizontal and vertical versions of each. For mobile screens, tabs are automatically turned into an accordion.

View Tabs Plugin

UI Widgets

Button States

Gives buttons states and event handlers. Buttons can be activated, deactivated, or disabled.

View Button States Plugin

Notifications

Creates customizable notification bars that can be event or time-based.

View Notifications Plugin

Reveal

Provides an easy way to toggle the visibility of any element with a trigger, with an optional slide transition.

View Reveal Plugin

Social Share

Sharing and voting widgets for up to 5 popular social media sites. Provides 6 different display options for the widgets.

View Social Share Plugin

Tooltips

2 different tooltip styles with customizable event triggers, orientations, offsets and transitions.

View Tooltips Plugin
