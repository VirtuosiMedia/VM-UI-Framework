Find a bug?

You can report it via our GitHub issue tracker. When you submit an issue, please help us out by taking the following steps:

  • Check to see if the issue has already been reported.
  • Record your browser, browser version, and operating system.
  • Provide sample code that will reproduce the bug. Please make the code as simple as possible to ensure there aren't other factors in play.
  • Screenshots are also helpful.
